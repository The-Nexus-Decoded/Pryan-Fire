# hughs-forge

Hugh the Hand's domain. Trading algorithms, financial connectors, and market analysis tools.

Deploys to: **ola-claw-trade** (Hugh)

## Contents

- `services/` — Trading services (Meteora DLMM SDK, risk manager, trade pipeline)
- `config/` — Service configuration templates (no secrets — use .env for actual values)

## What's NOT tracked

- `strategies/` — Trading strategies are private, never committed to git
- `.env` — API keys, wallet addresses, RPC URLs
